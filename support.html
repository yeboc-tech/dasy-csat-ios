1. 앱 소개

Dasy CSAT는 한국 고등학생을 위한 수능(대학수학능력시험) 준비 교육 앱입니다. 인터랙티브 PDF 보기, OMR 마킹, 주석 도구 등을 제공하여 효과적인 학습을 지원합니다.

2. 주요 기능

  • PDF 문서 보기: 수능 시험 자료를 인터랙티브하게 탐색
  • 주석 도구: 문서에 그림, 메모, 하이라이트 추가
  • OMR 마킹: 답안지 마킹 및 자동 채점
  • 로컬 저장: 모든 작업 내용을 기기에 안전하게 저장
  • 교육 콘텐츠: 한국교육과정평가원(KICE) 공식 자료 활용

3. 자주 묻는 질문 (FAQ)

Q: 앱을 처음 사용하는데 어떻게 시작하나요?
A: 앱을 실행하면 홈 화면이 나타납니다. "문서 열기" 버튼을 탭하여 수능 시험 자료를 선택하고 학습을 시작하세요.
Q: 내가 작성한 주석과 그림은 어디에 저장되나요?
A: 모든 사용자 생성 콘텐츠는 기기에 로컬 저장됩니다. 인터넷 연결 없이도 언제든지 접근할 수 있습니다.
Q: OMR 마킹 기능은 어떻게 사용하나요?
A: 답안지 페이지에서 OMR 마킹 도구를 선택하고, 해당하는 답안 번호를 탭하여 마킹하세요. 자동으로 채점 결과를 확인할 수 있습니다.
Q: 데이터를 백업하거나 다른 기기로 옮길 수 있나요?
A: 현재 앱은 로컬 저장 방식을 사용하므로, 기기 간 데이터 동기화는 지원하지 않습니다. 중요한 주석은 스크린샷으로 백업하시는 것을 권장합니다.
Q: 앱이 느리거나 멈추는 경우 어떻게 해야 하나요?
A: 앱을 완전히 종료하고 다시 실행해보세요. 문제가 지속되면 기기를 재시작하거나 앱을 재설치해보세요.
Q: 개인정보가 수집되나요?
A: 아니요. 앱은 이름, 이메일 등 개인정보를 수집하지 않습니다. 모든 데이터는 기기에만 저장됩니다.

4. 문제 해결 가이드

4.1 앱이 실행되지 않는 경우

  1. 기기를 재시작합니다
  2. 앱을 완전히 종료하고 다시 실행합니다
  3. iOS 버전이 최신인지 확인합니다
  4. 필요한 경우 앱을 재설치합니다

4.2 PDF가 열리지 않는 경우

  1. 인터넷 연결을 확인합니다
  2. 앱을 다시 실행합니다
  3. 기기 저장 공간이 충분한지 확인합니다
  4. 다른 문서를 시도해보세요

4.3 주석이 저장되지 않는 경우

  1. 기기 저장 공간을 확인합니다
  2. 앱을 완전히 종료하고 다시 실행합니다
  3. 주석 도구가 올바르게 선택되었는지 확인합니다
  4. 문서를 다시 열어보세요
